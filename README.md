# CustomCircleView
Custem Circle  View

  Step 1. Add it in your root build.gradle at the end of repositories:
    \n
  allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  
  Step 2. Add the dependency
  dependencies {
	        implementation 'com.github.chy116449859:CustomCircleView:Tag'
	}
